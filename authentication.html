Login

This page will demonstrate some basic front-end authentication. Not great security, however some security.

When the user clicks "Log in", the username and passwords entered will be read by the authentication() function and checked against the correct values below.

If the user enters correctly, the output is updated with "Success" in green.

If the user enters incorrectly, the output is updated with "Failed" in red.

The correct username is admin.
The correct password is admin.

Note to students: In your tutorial, you should also discuss and/or demonstrate why this is not ideal security, and some other possibilities for better security of usernames and passwords.
